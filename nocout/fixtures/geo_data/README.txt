################################## Multiple file fixture #####################################

Order in which we need to run fixtures:
1. country.json
2. state.json
3. city.json
4. stategeoinfo.json

Command to run fixture:
>>> python manage.py loaddata <file_name>
for e.g.
>>> python manage.py loaddata country.json




################################## Single file fixture #####################################

Generated by following command:
>>> ./manage.py dumpdata country state city stategeoinfo --indent 2 > initial_default_data.json

Execute by folowing command:
>>> python manage.py loaddata country_state_city_stategeoinfo.json
