################################## Multiple file fixture #####################################

Order in which we need to run fixtures:
1. command_default.json
2. service_default.json
3. machine_default.json
4. site_instance_default.json
5. device_group.json
6. device_default.json
7. user_group_default.json
8. auth.json
9. user_profile_default.json
10. inventory_default.json
11. organization_default.json

Command to run fixture:
>>> python manage.py loaddata <file_name>
for e.g.
>>> python manage.py loaddata auth.json
